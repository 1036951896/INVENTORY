🆘 Solución Rápida de Problemas

  • ❌ Backend no inicia:
    cd backend && npm install && npm run start:dev
  • ❌ Frontend no inicia:
    cd frontend && npm install && npm run dev
  • ❌ Login no funciona:
    Limpia cache (CTRL+SHIFT+DEL) y recarga (F5)
  • ❌ No redirige a /admin:
    Abre DevTools (F12) y revisa: localStorage.getItem('admin-usuario')
