# 📚 Legacy Code - Código Anterior a React

Esta carpeta contiene el **código HTML, CSS y JavaScript original** del proyecto, anterior a la migración a React.

## ¿Por qué está aquí?

- **Propósito histórico**: Referencia del código inicial antes de la refactorización moderna
- **Documentación**: Ejemplos de diseño, estructura y lógica anterior
- **No está en producción**: Render está desplegando la versión React moderna en `frontend/`
- **Referencia visual**: Para entender la evolución del proyecto

## Estructura

```
html-css-js-inicial/
├── html/           # Páginas HTML originales
├── css/            # Estilos CSS originales
├── js/             # JavaScript vanilla original
├── assets/         # Recursos originales
├── images/         # Imágenes del sitio
└── data/           # Datos estáticos JSON
```

## Código Activo

El código que **está en producción** es:

- **Frontend**: `frontend/` (React + Vite - Render)
- **Backend**: `backend/` (NestJS - Render)
- **Config**: `nginx/` (Nginx production)

**Nota**: Esta carpeta se conserva solo con fines de referencia y documentación histórica del proyecto.
